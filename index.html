hexo是什么

hexo是基于Node.js的博客框架，支持 Markdown，可部署到 GitHub Pages。
由于是基于Node.js的，因此对前端程序员天然友好。
官网：https://hexo.io/zh-cn/
安装部署教程网上较多，在此不再赘述，仅记录安装部署过程中遇到的坑及解决办法。
