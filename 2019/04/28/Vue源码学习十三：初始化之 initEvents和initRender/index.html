上面的代码主要作用就是在 Vue 实例对象上定义两个属性：vm.$attrs 以及 vm.$listeners。这两个属性在 Vue 的文档中是有说明的，由于这两个属性的存在使得在 Vue 中创建高阶组件变得更容易，感兴趣的同学可以阅读 探索Vue高阶组件。

我们注意到，在为实例对象定义 $attrs 属性和 $listeners 属性时，使用了 defineReactive函数，该函数的作用就是为一个对象定义响应式的属性，所以 $attrs 和 $listeners 这两个属性是响应式的，至于 defineReactive 函数的讲解，我们会放到 Vue 的响应系统中讲解。
