eslint进行代码检查

前言

最近项目进行到收尾阶段，公司对于代码质量抓的比较严，后端一般使用sonar进行代码扫描。而JavaScript是弱类型语言，其实很有扫描检查的必要。
sonar支持JS和typeScript的扫描，不支持Vue。因此改用eslint进行代码扫描。
平常在开发中，都是嫌麻烦把eslint关掉的。。

安装依赖

除了安装eslint以外，还需安装eslint-plugin-vue插件来检查Vue代码。
直接-g吧。
npm install eslint eslint-plugin-vue -g

代码扫描

项目根目录运行eslint --ext .js,.vue src。
或者将运行结果输出到文件eslint --ext .js,.vue src > log.txt。
